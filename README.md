# Expense Tracker App

This is a simple expense tracker app built with React and Redux.

## Installation

1. Clone the repo

```bash
git clone https://github.com/rudrakshi99/Expense-tracker-app.git
```

2. Install NPM packages

```bash
npm install
```

3. Run the app

```bash
npm start
```

## Tech Stack

- React
- Redux
